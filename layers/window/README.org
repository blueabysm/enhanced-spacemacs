* Description
Better window management, inspired by [Steve Purcell](https://github.com/purcell)
  - `C-x |` split window vertically and instead
  - `C-x -` split window horizontally and instead

* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =window= to the existing =dotspacemacs-configuration-layers= list in this
file.

* Key bindings

| Key Binding | Description    |
|-------------+----------------|
| ~SPC x x x~ | Does thing01   |
